# dotfiles
My dotfiles for WindowMaker  
## Other configs:
[XTerm](https://github.com/minimumattic/xterm)  
## Dependencies:
[windowmaker](https://www.windowmaker.org/) (window manager)  
[sxhkd](https://github.com/baskerville/sxhkd) (hotkey daemon)  
### Dockapps
[wmacpi](https://www.dockapps.net/wmacpi) (battery stats)  
[asmon](https://www.dockapps.net/asmon) (system monitor)  
[wmitime](https://www.dockapps.net/wmitime) (clock)  
[wmweather](https://www.dockapps.net/wmweather) (weather)  
[wmix](https://www.dockapps.net/wmix) (audio mixer)  
[wmeyes](https://www.dockapps.net/wmeyes) (xeyes)
## Directories:
* `GNUstep/*` to `~/GNUstep/`
* `sxhkd/*` to `.config/sxhkx/`  
* `.Xresources` to `~/.XResources`  
* `wallpaper/*` to `~/wallpaper/`
